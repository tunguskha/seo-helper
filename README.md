# SEO helper

A little script that checks if your SEO is good.

It checks if the title, the description and the images are good.

That's it.

---


### CDN

```html
<script src="https://cdn.jsdelivr.net/gh/tunguskha/seo-helper@latest/seo-helper.min.js"></script>
```
